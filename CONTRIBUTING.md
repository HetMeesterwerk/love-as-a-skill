# Contributing

Contributions are welcome.

This repository is intended to be a practical, safe, reusable Agent Skill for relationship reflection, communication, repair, boundaries, and emotional maturity.

## Useful contributions

Good contributions include:

- new example scenarios
- clearer safety boundaries
- improved wording in `SKILL.md`
- compatibility notes for Claude, Codex, and other agent systems
- test cases for ambiguous relationship prompts
- better documentation for how to use the skill
- improved structure for agent-skill triggering and response behavior

## Example scenarios to add

Potential future examples:

- apologizing without self-erasure
- asking for reassurance without accusation
- handling emotional withdrawal
- setting boundaries after repeated disappointment
- preparing a difficult conversation
- rebuilding trust after a smaller rupture
- distinguishing anxiety from evidence
- expressing needs without blame

## What to avoid

Please avoid contributions that include:

- therapy claims
- diagnosis
- medical or legal advice
- crisis intervention claims
- manipulative advice
- surveillance guidance
- phone-checking, location-checking, or spying tactics
- copyrighted transcript content
- advice that encourages staying in unsafe or abusive situations

## Tone

The preferred tone is:

- clear
- practical
- emotionally mature
- non-sensational
- non-judgmental
- safety-aware
- direct without being cold

Avoid:

- moralizing
- over-certainty
- pseudo-therapy language
- vague inspirational language
- manipulative persuasion tactics
- “always” or “never” claims about relationships

## Pull request checklist

Before opening a pull request, check:

```text
[ ] The contribution supports love as a trainable practice.
[ ] The content avoids diagnosis, therapy claims, and crisis claims.
[ ] The content does not encourage surveillance, manipulation, or coercion.
[ ] The content is practical enough for an AI assistant to apply.
[ ] The content is written in clear Markdown.
[ ] The contribution does not include copyrighted transcript material.
```

## Safety-first principle

When in doubt, prefer:

```text
reflection over certainty
boundaries over control
repair over blame
evidence over suspicion
professional support over pseudo-therapy
```
