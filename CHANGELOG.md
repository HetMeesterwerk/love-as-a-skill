## [1.0.0] - Initial public release

### Added

- Initial `SKILL.md`
- Five-layer reflection model:

```text
Facts → Feelings → Meaning → Pattern → Skill
```

- Safety boundaries for non-therapy, non-diagnostic, non-surveillance use
- Example workflow: conflict repair after an argument
- Example workflow: jealousy without surveillance
- Example workflow: compatibility versus communication reflection
- MIT License
- Public GitHub repository

### Notes

This first release defines the core skill behavior:

- treat love as a trainable practice
- separate facts from interpretations
- support repair-focused communication
- distinguish boundaries from control
- avoid surveillance, coercion, diagnosis, and therapy claims
